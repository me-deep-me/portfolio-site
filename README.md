# Mattia Erigoni Portfolio

Personal portfolio for [mattiaerigoni.com](https://mattiaerigoni.com).

The site presents operational software projects across logistics, planning, data quality, process digitalization and private AI workflows. It is built with Next.js, React, Tailwind CSS and Framer Motion.

## Local Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Checks

```bash
npm run lint
npx tsc --noEmit --pretty false
npm run build
```

## Notes

- Generated build folders, packaged executables and Python cache files are intentionally ignored.
- Project demos are linked from the portfolio and open in popup windows when supported by the browser.
- The production branch is `main`; the active working branch is `v2`.
